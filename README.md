# BEM-VINDO AO CHAT_GPSIR
<p>Este é um projeto bem simples mas bem executado que reproduz o chat gpt de forma minimalista e direta, o chat foi criado com javascript vanilla.</p>
<img src="./Assets/chat-preview.png" style="margin-left: 30%">

## Funções
- Capacidade de responder perguntas da mesma forma que o chat_gpt v3.5
- Continuidade de conversa temporaria

## Importante
- O chat tem continuidade de conversa por sessão, reconhecendo a mesma conversa até que a aba seja fechada. 
- Não mande mensagem que contém violencia kkk
